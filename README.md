# Data Science Projects - VŠE Prague 

This repository contains my academic projects and individual tasks developed during my Data Science studies at the **University of Economics and Business (VŠE)** in Prague.

##  Key Skills Demonstrated
* **Data Cleaning & Manipulation:** Handling missing values and outliers using R.
* **Exploratory Data Analysis (EDA):** Statistical summaries and trend analysis.
* **Data Visualization:** Creating meaningful plots (histograms, boxplots, scatter plots) to communicate insights.
* **Database Connectivity:** Fetching and analyzing data directly from MySQL databases.

##  Project Highlights
* **Movie Ratings Analysis:** SQL-based exploration of classic vs. modern cinema trends.
* **Biometric Study (Davis):** Analyzing the relationship between reported and measured weight/height.
* **PISA International Scores:** Correlation analysis between math and reading performance across countries.
* **Salary Distribution:** Comparing historical salary data (2013 vs. 2023) using statistical distributions.
